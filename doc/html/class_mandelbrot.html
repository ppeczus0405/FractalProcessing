pair< int, tuple< Complex, Complex, Complex > > Mandelbrot::getIterationsAndOrbit ( const Complex &  c)
overridevirtual

Get number of iterations and last 3 elements of orbit trap.

Parameters
c- complex number
Returns
pair consist of number of iterations and 3 last elements of orbit trap where:
  • last element is first iteration value exceeding bailout value,
  • second element is iteration value before last element,
  • first element is iteration value before second element,
    Number of iterations denote how many iterations needs algorithm perform to fall into the orbit trap. If the algorithm don't fall into the orbit trap and maximum number of iterations was reached then maximal number of iterations should be returned
